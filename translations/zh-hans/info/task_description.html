在本任务中，您需要按照以下步骤将嵌套密码字典转换为纯文本。这种转换与Cipher Dict 任务中的转换相反。

  1. 从最上面的字典开始，提取密钥并按正确顺序排列成一个数字块。区块顺序应严格交替，第一个区块必须是升序。依次检查每个按键的值（也包括字典），以找到下一个区块，并在以后依次检查新的按键。

  2. 将所有成立的区块并入一个十进制数。

  3. 将十进制值转换为十六进制数，并将结果转换为字符串。

描述可能令人困惑，让我们看看示例。

输入：作为字典的密码数据。

输出：字符串形式的原始文本：字符串形式的原始文本。
